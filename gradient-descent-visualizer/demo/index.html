üíª Quick Start

Get started with the Gradient Descent Visualizer in just a few lines of code:

// Install via npm npm install gradient-descent-visualizer // Import and use import { createVisualizer } from 'gradient-descent-visualizer'; // Create visualizer instance const container = document.getElementById('my-container'); const visualizer = createVisualizer(container, { mode: '2d', // '2d' or '3d' initialFunction: 'quadratic', // Built-in function initialPosition: { x: 2, y: 2 }, // Starting point learningRate: 0.1, // Step size autoPlay: true // Start immediately }); // Add event listeners visualizer.onConvergence = (stats) => { console.log('Converged in', stats.iterations, 'steps'); };

That's it! The visualizer will handle everything else, including UI controls, animation, and performance optimization.
